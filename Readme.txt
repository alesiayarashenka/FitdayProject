--------------Чек-лист для проверки https://www.fitday.com/

--------Модуль Login
1. Логин с корректными username и password
2. Логин с некорректными username и password
3. Логин с пустым username
4. Логин с пустым password

--------Модуль Forgot Rassword
1.Изменение пароля с корректным username

--------Модуль Header
1. Проверка отображения баннера
2. Переход на главную страницу при нажатии на баннер
3. Отображение аватара и имени пользователя
4. Проверка значений выпадающего списка доступных пользователю страниц
5. Проверка значений выпадающего списка нотификаций

--------Модуль Forums
1. Проверка отображений табов Forums, Tools, New Posts
2. Проверка совпадений значений выпадающего списка таба Forums с темами форумов на странице
3. Проверка отображения кнопок User CP,	FAQ, Community,	Calendar, New Posts, Search, Quick Links, Log Out
4. Выход из приложения

--------Модуль Search
1. Поиск на странице форумов по веткам
2. Поиск на странице форумов по постам
3. Поиск по тэгам
4. Поиск по ключевому слову в заголовке с очищением поля
5. Поиск по username
6. Поиск с дополнительным опциями (количество ответов, время публикации, сортировка)
7. Поиск по новым постам

--------Модуль My profile
1. Проверка отображения табов About Me, Statistics, Social Profiles, Friends, Mentions
2. Заполнение, редактирование и удаление информации во вкладке About Me

--------Модуль Thread
1. Создание нового треда без названия
2. Создание нового треда с названием на кириллице
3. Создание треда с валидными значениями в полях, используя предпросмотр
4. Создание ответа на тред, используя предпросмотр
5. Создание быстрого ответа

--------Модуль Subscribe
1. Проверка подписок пользователя
2. Подписка на тред
3. Удаление подписки на тред
